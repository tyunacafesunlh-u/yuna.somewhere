# Daily Business Checklist App

業務用の日次チェックリストアプリケーションです。
スプレッドシート管理から移行し、タブレットやPCでのタッチ操作に最適化されています。

## 特徴
- **時間帯別リスト**: 9時、10時、15時、18時、19時など、業務フローに合わせた構成
- **曜日別表示**: 特定の曜日（「月曜のみ掃除」「木曜のみ段ボール」など）にのみタスクを表示
- **データ自動保存**: ブラウザを閉じてもチェック状況を保持（LocalStorage使用）
- **日次リセット**: 日付が変わると自動でチェックをクリアし、前日のデータを履歴として保存
- **レスポンシブ**: デスクトップでは多列表示、モバイルでは縦一列表示

## 使い方
1. `index.html` を開きます。
2. 業務に合わせてチェック項目をタップします。
3. リセットボタンで手動リセットも可能です。

## GitHub Pagesでの公開
このリポジトリをGitHub Pagesで公開することで、スタッフ全員がブラウザからアクセスできるようになります。
※注意: データは各端末のブラウザに保存されるため、スタッフ間でのチェック状況の同期は行われません。
